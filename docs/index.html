Functions

command (cmd, ...)
Runs a simple command, without capturing output. *Curried function.

Parameters:

  • cmd string Command to be run.
  • ... Parameters to the command, may be tables (if using $) or strings.

Returns:

  1. boolean Whether the command ran successfully or not.
  2. string Reason for exit ("exit" or "signal").
  3. number or nil If the reason for exit is "signal, this contains the signal.
capture (cmd, ...)
Runs a command and captures its output. Curried function. This function uses io.popen, which may not be availiable in all platforms.

Parameters:

  • cmd string Command to be run.
  • ... Parameters to the command, may be tables (if using $) or strings.

Returns:

    string Result of the command.
interact (cmd, ...)
Perform basic IO operations on a command. Curried function. The command structure returned only contains one field, command, which contains the final command.

Parameters:

  • cmd string Command to be run.
  • ... Parameters to the command, may be tables (if using $) or strings.

Returns:

    Command Command structure.
